# Potepass – Hundepasser-tjeneste

## Om prosjektet
Potepass er en digital tjeneste som kobler hundeeiere med hundepassere. Løsningen gjør det mulig å finne hundepassere, opprette bookinger og administrere. Prosjektet er utviklet som en del av Arbeidskrav 1–3, og følger smidig utviklingsmetodikk (Scrum).

## API og ressurser
Prosjektet bruker følgende API-ressurser:
- users
- petSitters
- bookings
Alle sider skal støtte full CRUD-funksjonalitet (Create, Read, Update, Delete).

## Sider og ansvarsfordeling

### Brukerprofil – Min profil
**API:** users  
**Ansvar:** Student A  

Funksjonalitet:
- Vise brukerens informasjon
- Opprette og redigere bruker
- Legge til, redigere og slette hunder
- Slette brukerprofil

### Finn hundepasser
**API:** petSitters  
**Ansvar:** Student B  

Funksjonalitet:
- Vise liste over hundepassere
- Enkel filtrering (sted / pris – hvis tid)
- Vise detaljer om valgt hundepasser
- Opprette, redigere og slette hundepassere

### Mine bookinger
**API:** bookings  
**Ansvar:** Maryna Tryus 

Funksjonalitet:
- Opprette og redigere booking
- Vise egne bookinger
- Endre status (pending / approved / rejected)
- Slette booking


## Arbeidsmetode
Prosjektet planlegges og følges opp i GitHub Projects ved hjelp av Kanban-board.
User stories og oppgaver oppdateres fortløpende med status (Todo, In progress, Done).

Sprintene er på 2 uker, med korte standups og retrospektiv etter hver sprint.
